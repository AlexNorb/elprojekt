# Bank Gothic Light Font Instruktioner

Kopiera dina Bank Gothic Light font-filer till `public/fonts/` mappen.

## Nödvändiga filer:
- `bank-gothic-light.woff2` (rekommenderas)
- `bank-gothic-light.woff` (fallback)

## Om du har andra format (.ttf, .otf):
Du kan konvertera dem till .woff2 och .woff på:
- https://transfonter.org/
- https://cloudconvert.com/ttf-to-woff2

## Efter kopiering:
Kör `npm run dev` igen för att se förändringarna.

Fonten kommer att laddas från `/fonts/` katalogen och användas för alla element med `font-display` klassen (logotypen, rubriker, knappar).
