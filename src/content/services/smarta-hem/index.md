Ett smart hem handlar inte bara om teknik, utan om att **förenkla din vardag**.

Tänk dig att belysningen automatiskt dimras ner på kvällen, eller att **"Hejdå"-knappen** vid dörren släcker hela huset och larmar på.

Vi installerar system som **KNX, Plejd och Wiser** som låter dig styra belysning, värme och solskydd direkt från mobilen.

<small>*Jag kan hjälpa dig på Tjörn, Orust, Stenungsund, Uddevalla, Kungälv och omkringliggande områden.*</small>
